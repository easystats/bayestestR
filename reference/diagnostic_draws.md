# Diagnostic values for each iteration

Returns the accumulated log-posterior, the average Metropolis acceptance
rate, divergent transitions, treedepth rather than terminated its
evolution normally.

## Usage

``` r
diagnostic_draws(posterior, ...)
```

## Arguments

- posterior:

  A `stanreg`, `stanfit`, `brmsfit`, or `blavaan` object.

- ...:

  Currently only used for models of class `brmsfit`, where a `variable`
  argument can be used, which is directly passed to the
  [`as.data.frame()`](https://rdrr.io/r/base/as.data.frame.html) method
  (i.e., `as.data.frame(x, variable = variable)`).

## Examples

# \donttest{
set.seed(333)

if (require("brms", quietly = TRUE)) {
  model <- suppressWarnings(brm(mpg ~ wt * cyl * vs,
    data = mtcars,
    iter = 100, control = list(adapt_delta = 0.80),
    refresh = 0
  ))
  diagnostic_draws(model)
}
